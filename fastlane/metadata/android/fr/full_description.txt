Par défaut, les numéros ne figurant pas dans vos contacts sont bloqués.

En option, vous pouvez autoriser :
* Les numéros que vous avez appelés
* Les numéros dans les groupes sélectionnés (ex : numéro locaux, non surtaxé)
* N appel enregistré du même numéro dans les T dernières minutes
* Les numéros de mobile des textos reçus
* Et d'autres...

Si l'application rejette les appels de contacts sous Android 10, autorisez manuellement l'accès aux
contacts dans Infos de l'appli → Autorisations.

Autorisations :
* CALL_SCREENING - Bloquer ou autoriser les appels.
* READ_CALL_LOG - Vérifie les numéros que vous avez appelé et décompter les fois où un numéro vous a appelé en T minutes.
* READ_SMS - Vérifier les numéros auquel vous avez envoyé un message.
* NOTIFICATION_LISTENER - Rechercher des numéros de téléphone mobile dans les messages entrants.
* READ_PHONE_STATE - Vérifier sur quelle sim le numéro appel.
* RECEIVE_BOOT_COMPLETED - démarrer le service de blocage des numéros après redémarrage de l'appareil.
* READ_CONTACTS - Vérifier qu'un numéro est présent dans les contacts sous Android 10 ou bloquer leurs appels.

Ceci est un logiciel libre et gratuit.
